# TDoc
A python documentation generator.


### Installation

```
pip install tdoc
```
[pypi](https://pypi.python.org/pypi/tdoc)


### Generate the markdown documentation.

```
tdoc generate --folder ./testfiles
```

### Run a local webserver to view your documentation.

```
tdoc serve
```
